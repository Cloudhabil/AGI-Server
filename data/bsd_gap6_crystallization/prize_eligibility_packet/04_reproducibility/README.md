## Usage

### Run All Verifications

```bash
python verify_bsd.py --all
```

### Verify Specific Vector

```bash
python verify_bsd.py --vector euler_systems
python verify_bsd.py --vector derived_selmer
python verify_bsd.py --vector infinity_folding
```

### Verify Specific Curve

```bash
python verify_bsd.py --curve 389a1    # Rank 2
```

## Verification Vectors

### 1. Euler Systems (`euler_systems`)

Constructs higher-rank Euler systems and verifies the control theorem:
- Computes Kato elements for rank 1
- Constructs wedge products for rank >= 2
- Verifies Selmer group bounds

### 2. Derived Selmer (`derived_selmer`)

Verifies virtual dimension equals algebraic rank:
- Computes H^0, H^1, H^2 of Selmer complex
- Applies Tate duality
- Checks vdim = rank

### 3. Infinity Folding (`infinity_folding`)

Implements the p-adic regulator computation:
- Applies folding transform to divergent series
- Computes p-adic heights
- Verifies regulator formula
