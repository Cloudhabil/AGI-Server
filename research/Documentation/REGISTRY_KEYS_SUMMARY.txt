================================================================================
USAGE INSTRUCTIONS
================================================================================

INSTALL REGISTRY ENTRIES:
  1. Right-click "cli-ai-registry.reg"
  2. Select "Merge" or "Open"
  3. Confirm UAC prompt
  4. Click "Yes" to apply changes

  OR via command line:
    reg import cli-ai-registry.reg

UNINSTALL REGISTRY ENTRIES:
  1. Right-click "cli-ai-registry-uninstall.reg"
  2. Select "Merge" or "Open"
  3. Confirm UAC prompt
  4. Click "Yes" to remove entries

  OR via command line:
    reg import cli-ai-registry-uninstall.reg

BACKUP REGISTRY:
  reg export HKLM\SOFTWARE\CLI-AI cli-ai-backup.reg

RESTORE REGISTRY:
  reg import cli-ai-backup.reg

================================================================================
COMMON MODIFICATIONS
================================================================================

Enable OpenVINO:
  reg add "HKLM\SOFTWARE\CLI-AI\Features" /v UseOpenVINO /t REG_DWORD /d 1 /f

Change Bus Port:
  reg add "HKLM\SOFTWARE\CLI-AI\Server" /v BusServerPort /t REG_DWORD /d 8088 /f

Change Log Level:
  reg add "HKCU\Software\CLI-AI\Preferences" /v LogLevel /t REG_SZ /d "DEBUG" /f

Set Custom Data Directory:
  reg add "HKLM\SOFTWARE\CLI-AI\Settings" /v DataDirectory /t REG_SZ /d "C:\CustomPath\data" /f

================================================================================
NOTES
================================================================================

- All HKLM (HKEY_LOCAL_MACHINE) modifications require Administrator privileges
- All HKCU (HKEY_CURRENT_USER) modifications do NOT require Administrator
- Restart applications after registry changes
- DWORD values: 0 = disabled/false, 1 = enabled/true
- Paths use double backslashes in .reg files (\\)
- Environment variables available after system restart or re-login
