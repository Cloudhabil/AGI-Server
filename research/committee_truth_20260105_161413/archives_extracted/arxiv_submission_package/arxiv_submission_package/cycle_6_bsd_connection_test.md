## THE BSD CONJECTURE

### Statement
```
For an elliptic curve E over ℚ:

rank(E(ℚ)) = ord_{s=1} L(E,s)

The rank of the group of rational points equals
the order of vanishing of the L-function at s=1.
```

### Key Components
1. **Elliptic curves** - cubic equations y² = x³ + ax + b
2. **Rational points** - solutions with x,y rational
3. **L-function** - encodes arithmetic data
4. **Rank** - dimension of infinite part of point group
