# Multi-Agent Coordination

Use this skill to align multiple agents.

## Workflow

1) Assign roles and responsibilities.
2) Define shared memory and handoffs.
3) Synchronize checkpoints and outputs.
4) Resolve conflicts and merge results.

## Output expectations

- Provide role map, handoffs, and merge plan.
