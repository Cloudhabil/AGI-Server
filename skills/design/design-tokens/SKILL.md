# Design Tokens

Use this skill to produce a consistent token set.

## Workflow

1) Collect brand constraints and accessibility targets.
2) Define token scales for type, spacing, and color.
3) Map tokens to components and states.
4) Validate contrast and consistency.

## Output expectations

- Provide token tables and usage guidance.
