# CLI AI Mindmap UI

This UI provides a 3D skill tree / mindmap visualization used by the GPIA documentation and demos.

## Development

```bash
cd ui/cli-ia-mindmap
npm install
npm run dev
```

The dev server runs at <http://localhost:5173> by default.

## Build

```bash
npm run build
```

## Notes

- The UI is intended for local visualization and does not require backend services.
- For the doc-facing skill tree, see `docs/README.html`.
